Movie API

Get list of all movies '/movies' GET Returns json file list of movies
Get data about a movie '/movies/:title' GET Returns json data for a specific movie Example: The Shining
Get data about movie's genre '/movies/genres/:title' GET Returns json data about a movies genre Example: Horror
Get data about a movie's director 'movies/director/:name' GET Returns json data about a movies director Example: Stanley Kubrick
Allows new user to register '/users' POST Adds new user to json object Example: New user Mike was added
Allows user to update username '/users/:username' PUT Returns updated username data Example: Username Mike was changed to Mike1
Allows users to add movie to favorites '/users/:username/movies/:movieId/favorites' POST Returns text confirming that a movie was successfully added Example: The Shining was added to your favorites
Allows user to delete a movie from favorites '/users/:username/favorites/:movieId' DELETE Returns text confirming that a movie was successfully deleted Example: The Shining was removed from your favorites
Allows user to unregister '/users/:username' DELETE Removes a user from json object You account was successfully deleted

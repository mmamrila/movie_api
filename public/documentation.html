Movie API

Homepage '/' GET Returns text "Welcome to MyFlix!"
Get list of all movies '/movies' GET Returns json file list of movies
Get data about a movie '/movies/:Title' GET Returns json data for a specific movie Example: The Shining
Get data about a genre '/genre/:Name' GET Returns json data about a genre Example: Horror
Get data about a director '/director/:Name' GET Returns json data about a director Example: Stanley Kubrick
Get list of all users '/users' GET Returns json file list of users
Allows new user to register '/users' POST Adds new user to json object Example: New user Mike was added
Allows user to update username '/users/:Username' PUT Returns updated username data Example: Username Mike was changed to Mike1
Allows users to add movie to favorites '/users/:Username/movies/:Title' POST Returns text confirming that a movie was successfully added Example: The Shining was added to your favorites
Allows user to delete a movie from favorites '/users/:Username/movies/:Title' DELETE Returns text confirming that a movie was successfully deleted Example: The Shining was removed from your favorites
Allows user to unregister '/users/:Username' DELETE Removes a user from json object You account was successfully deleted
